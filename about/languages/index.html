Languages

The following languages are currently included in the scope of the Histoires pour Gabon project:

  • French
  • Fang
  • Myene / Omyene
  • Punu / Yipunu
  • Nzebi / Yinjebi
  • Mbere / Mbede / Mbete / Limbede
  • Shira / Sira / Eshira
  • English
  • Portuguese
  • Spanish

Please note that most translations and recordings are not yet complete.

We are always looking for translators! If you speak one of the above languages and would like to help us translate or record stories, please contact us at globalafricanstorybook [at] gmail [dot] com
